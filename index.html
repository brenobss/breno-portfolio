Olá.
Meu nome é Breno Barreto.

Sou um Desenvolvedor Web Fullstack com o objetivo de solucionar problemas do mundo real através do mundo digital.

Tenho experiência em desenvolvimento front-end com o trio html, css e javascript, com a biblioteca ReactJs entre outros, como Bootstrap e Angular. Já no server-side (back-end), tenho ambientação com java, spring boot, php, laravel, nodejs, express, entre tantos outros, também sei mexer em banco de dados, principalmente os SQL, MySQL e MariaDB.
